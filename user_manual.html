1. Introduction (บทนำ)

This document provides a step-by-step guide for using the DHL Express Shipment Creation (Backup Solution). This tool allows you to create shipments, generate Air Waybills (AWB), and print invoices manually during system maintenance.

URL: https://sbs-e2e.vercel.app/ (Test Environment)

2. Getting Started (การเริ่มต้นใช้งาน)

2.1 Landing Page

When you access the system, you will see the Home Page.

  1. Language Selection: You can switch between Thai and English using the flag/language icon in the top right corner.
  2. Consent: You must check the box "I agree to the Terms of Use" to proceed.
  3. Start: Click the "Create Shipment" (สร้างใบนำส่ง) button.

3. Creating a Shipment (ขั้นตอนการสร้างใบนำส่ง)

The shipment creation process consists of 4 Main Steps:

Step 1: Shipment Details (ข้อมูลการจัดส่ง)

  • Shipment Date: Select the date of shipment.
  • Shipment Type: Choose between Documents (เอกสาร) or Packages (พัสดุ).
    • If you select Packages, you will need to declare the Value and Currency.
  • Payment: Select who pays for the shipping (Shipper or Receiver).

Step 2: Addresses (ที่อยู่)

  • Shipper Details (ผู้ส่ง):
    • Fill in Company/Name, Phone, and Full Address.
    • Tip: The system auto-saves your shipper details for next time.
  • Receiver Details (ผู้รับ):
    • Select Country first.
    • Enter Postal Code and City.
    • Fill in Company/Name and Phone.

Step 3: Package Details (รายละเอียดพัสดุ)

  • Enter the Weight (kg) and Dimensions (L x W x H cm) for each piece.
  • You can add multiple pieces if needed.
  • Description: Provide a clear description of the contents (e.g., "Spare Parts", "Contracts").

Step 4: Summary & Services (สรุปและบริการเสริม)

  • Protect Your Shipment: Check if you want to add insurance.
  • Pickup: Toggle "Schedule a Pickup" if you need a courier to collect the shipment.
  • Review the Shipment Summary on the right side.

4. Confirmation & Printing (การยืนยันและพิมพ์เอกสาร)

Once you click "Confirm Shipment", the system will generate your AWB.

Success Screen

You will see a confirmation page with:

  • AWB Number: Your tracking number (e.g., 1234567890).
  • Download Buttons:
    • Download Label: The AWB PDF.
    • Download Invoice: The Commercial Invoice (if applicable).
    • Download Receipt: The shipment receipt.

Printing Instructions

  1. Click Download Label.
  2. Open the PDF and print 2 copies (one for the package, one for the courier).
  3. Sign the invoice if required.

5. Troubleshooting (การแก้ปัญหาเบื้องต้น)

  • "Fill all fields" error: Check for any red boxes indicating missing information.
  • Cannot click Next: Ensure the "Consent" box is checked or all required fields in the current step are full.
  • Clear Data: Use the "Clear History" button in the menu to reset the form.
